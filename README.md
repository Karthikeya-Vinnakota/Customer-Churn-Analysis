# Customer-Churn-Analysis
Customer Churn Analysis using Excel, Python, and Power BI to identify factors affecting customer retention.

## Tools Used
- Excel
- Python (Pandas, Matplotlib, Seaborn)
- Power BI

## Key KPIs
- Total Customers
- Churn Rate
- Average Monthly Charges
- Customer Tenure

## Key Insights
• Customers with monthly contracts have higher churn  
• Higher monthly charges increase churn probability  
• Customers with low tenure are more likely to leave  

## Dashboard
Power BI dashboard showing churn trends and retention insights.
